smart contract deployment procedure:

1.compile contracts:
  cmd : ~/cbt_smart_contracts$ truffle compile

2.prepare etheruem account to use for deployment (deployment account)
 
3.prepare multisig owners accounts , list the addresses in ~/cbt_smart_contracts/deployment/owners.json

4. generate multisig bytecode to deploy 
    cmd: ~/cbt_smart_contracts/deployment$ node ./deploymenet Multisig
    the generated byte code will b save into ./MultiSigContract.bytecode

5.deploy multisig 
  deploy multisig byte code (saved in MultiSigContract.bytecode) using myetherwallet

  save contract address into MultiSigWallet.address
  
6.generate CommerceBlock Token byte code to deploy 
	cmd:  ~/cbt_smart_contracts/deployment$ node ./deployment deploy CBToken
	
	the script will use the multisig address saved in MultiSigWallet.address
	 the generated byte code will b save into ./CBTokenContract.bytecode

7. deploy CommerceBlock token 
  deploy CommerceBlock token bytecode  (saved in  ./CBTokenContract.bytecode) using myetherwallet
	save contract address into CBTokenContract.address
